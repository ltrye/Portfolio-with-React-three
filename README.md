# React three portfolio

A 3d website make with react-three-fiber.
Bundler: Webpack
Backend: Expressjs
Lib: Reactjs, React-three-fiber

# Getting Started

First, install all packages:

```bash
npm install
# 
npm run install-client
# 
npm run install-server
```

Run the app:
```bash
#In development
cd Client
npm run dev

#In production
npm run build
npm run start
```
